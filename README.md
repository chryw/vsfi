# vsfi
### Icon font generator
Original Bowtie icons (https://github.com/jimlamb/Bowtie) by @jimlamb

### Prerequisites
- nodejs 4.1+
- python 2.7

### Use
- clone repo
- npm install
- put *.svg format icons in assets/icons/ folder
- edit templates as needed
- edit gulp task name, icon name, file path, etc. in gulpfile.js as needed
- run gulp task
